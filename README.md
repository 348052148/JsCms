# JsCms
一个基于java web实现的CMS,有着不输于php 开发的cms系统的灵活性。

#介绍：
####采用原生j2ee开发。

#运行：
####使用myeclipse 打开，直接导出war包，将war包放到tomcat web跟目录，便可以在tomcat服务上运行。(前提是你有数据库字段)
####成功安装数据库后。执行服务器ip+war包名字/admin.php 便可进入·后台管理面板.前台展示，只需要服务器ip(或者域名)+war包/index.php便可进入
